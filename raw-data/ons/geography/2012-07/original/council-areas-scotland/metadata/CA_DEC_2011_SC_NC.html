Council Areas (Sco) Dec 2011 Names and Codes
Abstract: 
This file contains names and codes for Council Areas in Scotland as at December 2011.

The 32 single tier Council Areas that cover Scotland are responsible for all aspects of local government. They were created in April 1996 following a review of the local government structure in Scotland. They are made up of electoral wards. 

This file contains the new 9 character coding as well as the old 4 character coding. For more information, please use the following link to the Code History Database (CHD) - http://www.ons.gov.uk/ons/guide-method/geography/products/names--codes-and-look-ups/code-history-database/index.html
